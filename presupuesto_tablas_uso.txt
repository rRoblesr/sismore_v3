TABLA: pres_base_actividadesproyectos
  - Modelo:
    - app/Models/Presupuesto/BaseActividadesProyectos.php (protected $table = 'pres_base_actividadesproyectos')
  - Controlador (principalmente como “base generada” por el ETL de actividades/proyectos):
    - app/Http/Controllers/Presupuesto/ImporActividadesProyectosController.php
      - importarActualizar(): borra BaseActividadesProyectos (y detalle) si existe para esa importación
      - verificarBase(): busca BaseActividadesProyectos por importacion_id
      - descargarBaseProcesada(): busca BaseActividadesProyectos por importacion_id
      - eliminar(): borra BaseActividadesProyectos (y detalle) si existe para esa importación
  - Repositorios/Consultas:
    - app/Repositories/Presupuesto/BaseActividadesProyectosRepositorio.php
      - baseids_fecha_max(): query directo a pres_base_actividadesproyectos para obtener ids “última fecha”
      - listado_ejecucion(): join pres_base_actividadesproyectos con pres_base_actividadesproyectos_detalle
    - app/Repositories/Presupuesto/GobiernosRegionalesRepositorio.php
      - anios(), meses(): union con pres_base_proyectos para poblar filtros
      - tipos_gobiernosregionales(tipo=1): usa pres_base_actividadesproyectos + detalle
      - tipos_gobiernosregionales(else): usa union/diferencia contra pres_base_proyectos
    - app/Repositories/Presupuesto/ModificacionesRepositorio.php
      - usa pres_base_actividadesproyectos + detalle como fuente para comparativos (tipo=1 y tipo=3)
